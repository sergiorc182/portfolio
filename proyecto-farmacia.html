Sistema de Gestion para Farmacia POS

Java • Swing • POO • MySQL

Acceso seguro con una interfaz pensada para el uso diario en mostrador y administracion.

Descripcion del Proyecto

Sistema completo desarrollado en Java para la gestion integral de una farmacia. La aplicacion permite administrar inventario, ventas, caja, compras, clientes, proveedores y usuarios, manteniendo trazabilidad y control operativo.

Tecnologias Utilizadas

Java Swing MySQL POO

Caracteristicas Principales

  • Interfaz grafica desarrollada en Java Swing.
  • Sistema de login con roles y permisos.
  • Punto de venta con control de caja.
  • Gestion de productos, clientes y proveedores.
  • Control de stock y compras.
  • Historial y seguimiento de ventas.
  • Administracion de usuarios y roles.
  • Base de datos MySQL para persistencia y consultas.

Modulos del Sistema

  • Login: acceso y validacion de usuarios.
  • Dashboard: resumen visual del estado del sistema.
  • Punto de venta: operaciones de mostrador y tickets.
  • Compras y proveedores: gestion de abastecimiento.
  • Clientes y ventas: seguimiento comercial e historico.
  • Usuarios y roles: administracion de permisos.
