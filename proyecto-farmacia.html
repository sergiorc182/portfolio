Sistema de Gestion para Farmacia POS

Java, Swing y MySQL para mejorar ventas, caja, compras y control interno en una farmacia.

Acceso seguro y una experiencia de uso pensada para el trabajo diario en mostrador.

Stack principal
Java Swing MySQL POO

Problema que resuelve: dar orden a la operacion de una farmacia integrando ventas, compras, caja, clientes, proveedores y control de usuarios.

Enfoque tecnico: sistema POS de escritorio con modulos separados, persistencia de datos y flujos optimizados para atencion y gestion administrativa.

Ver codigo Volver a proyectos

Descripcion

Este sistema fue pensado para gestionar procesos frecuentes de una farmacia desde una sola aplicacion. Incluye operaciones de punto de venta, administracion de caja, compras, clientes, proveedores y seguimiento historico.

Funcionalidades clave

  • Login con control de roles y permisos.
  • Punto de venta con control de caja.
  • Gestion de compras y proveedores.
  • Modulo de clientes e historial de ventas.
  • Administracion de usuarios y roles.

Aprendizajes tecnicos

  • Desarrollo de interfaces de escritorio con Java Swing.
  • Persistencia y consultas sobre MySQL.
  • Diseno modular para separar responsabilidades.
  • Enfoque en productividad y claridad para el usuario final.
