Sistema de Gestion para Kiosco

Python • Tkinter • POO • MySQL

Interfaz moderna y segura para autenticacion de usuarios.

Descripcion del Proyecto

Sistema completo desarrollado en Python para la gestion integral de un kiosco o pequeno comercio. La aplicacion permite administrar inventario, realizar ventas, controlar stock, turnos de empleados y generar reportes para mejorar la operacion diaria.

Tecnologias Utilizadas

Python Tkinter MySQL POO

Caracteristicas Principales

  • Interfaz grafica intuitiva con Tkinter.
  • Sistema de login con roles de administrador y empleado.
  • CRUD completo para productos, clientes y proveedores.
  • Sistema de ventas con generacion de tickets.
  • Control de stock con alertas automaticas.
  • Reportes de ventas diarias, semanales y mensuales.
  • Backup automatico de la base de datos.
  • Sistema de busqueda avanzada de productos.
  • Gestion de multiples usuarios con permisos.

Estructura del Proyecto

El proyecto sigue una arquitectura modular con las siguientes capas:

  • login.py: ventana de autenticacion de usuarios.
  • main_app.py: aplicacion principal del sistema.
  • turno.py: gestion de turnos para empleados.
  • base de datos: configuracion y persistencia de informacion.
  • Modulos: clases de negocio como producto, venta, inventario y gastos.
